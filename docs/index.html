        작동 원리
        
3가지만 기억하세요

        

          ① 프로젝트 읽기 → ② 설치 목록에서 빼고 싶은 번호만 말하기 → ③ 맞는 폴더에 자동 설치
        

        

          

            
1

            
프로젝트 읽기

            
package.json, README 등으로 Next.js·Python 같은 스택을 파악합니다.

          

          

            
2

            
목록 확인

            
맞는 규칙·스킬 목록을 번호로 보여줍니다. 빼고 싶은 번호만 말하면 됩니다.

          

          

            
3

            
자동 설치

            
Cursor면 .cursor/, Codex면 .agents/skills/ 등 알맞은 폴더에 넣습니다.

          

        

        

          자세히: AI가 하는 8단계
          

          

            
1

            

              
🕵️프로젝트 분석

              
package.json, README 등을 읽고 Next.js, Python, Tailwind 같은 스택 키워드를 뽑아냅니다.

            

          

          

            
2

            

              
🧹기존 규칙 점검

              
이미 있는 규칙 중 outdated·충돌 나는 것을 찾아 제거 제안. 동의 없이 삭제 안 함.

            

          

          

            
3

            

              
📥레지스트리 검색

              
4개 공식 GitHub 레지스트리에서 스택에 맞는 규칙·스킬을 찾아 점수 매기고 목록 작성 (최대 50개).

            

          

          

            
4

            

              
⚙️자동 설치

              
목록에 이의 없으면 .cursor/rules/, .agents/skills/ 등 알맞은 폴더에 파일을 깔아줍니다.

            

          

          

            
5

            

              
🧬유지보수 리마인더

              
스택이 바뀌면 다시 setup 돌리라는 메타 규칙을 심어줍니다.

            

          

          

            
6

            

              
🛡️Git 안전

              
.gitignore에 .cursor/, .claude/ 등 에이전트 폴더를 자동 추가 (이미 있으면 건너뜀).

            

          

          

            
7

            

              
📋최종 리포트

              
뭘 설치했는지, 어디서 가져왔는지 표로 정리해서 보여줍니다.

            

          

          

            
8

            

              
🙋수정 루프

              
"이거 빼줘", "React 규칙 더 넣어줘" 같이 말하면 바로 반영합니다.

        FAQ
        
바이브코더 FAQ

        
코딩 잘 몰라도 OK. 자주 묻는 질문만 쏙쏙.


        

          

            
            

              네. 프롬프트 복사 → AI 채팅에 붙여넣기 → Enter만 하면 됩니다.
              AI가 파일 읽고, 다운로드하고, 폴더에 넣어줍니다. 터미널 명령어 칠 필요 없어요.
            

          

          

            
            

              됩니다. Claude Code, Windsurf, Copilot, Gemini CLI 등 18개 이상 도구를 지원해요.
              AI가 .claude/, .cursor/, .github/skills/ 같은 폴더를 자동으로 찾아서 설치합니다.
            

          

          

            
            

              동의 없이 삭제하지 않습니다. 낡거나 충돌 나는 규칙이 있으면 "이거 지울까요?"라고 먼저 물어봅니다.
              설치도 목록을 보여준 뒤, 빼고 싶은 항목만 말하면 됩니다.
            

          

          

            
            

              4개 공식 GitHub 레지스트리만 사용합니다:
              awesome-cursorrules,
              antigravity-awesome-skills,
              awesome-clinerules,
              awesome-cursor-skills.
              검증된 소스만 쓰고, 악성 override 문구는 설치 전에 제거합니다.
            

          

          

            
            

              스택이 크게 바뀌면 같은 프롬프트를 다시 돌리는 걸 추천합니다.
              AI가 새 스택에 맞는 규칙을 찾아주고, setup 완료 후 "스택 바뀌면 다시 실행" 리마인더도 심어줍니다.
            

          

          

            
            

              지금 대화 중인 제품(Cursor 채팅, Claude Code 등)과 프로젝트 폴더(.cursor/, .claude/ 등)를 자동으로 감지합니다.
            

          

          

            
            

              AI가 보여준 번호만 말하면 됩니다. 예: 「3번 빼줘」
            

          

          

            
            

              정상입니다. Codex 공식 경로는 .agents/skills/와 AGENTS.md입니다.
            

          

          

            
            

              .cursor/, .claude/, .antigravity/ 같은 AI 설정 폴더가 git에 올라가지 않도록 자동 추가합니다.
              팀 repo에 개인 AI 설정이 섞이는 걸 방지해요. 이미 있으면 건너뜁니다.
